변수 선언할 때 쓰는 키워드
기본 형태: let variableName = 'value'
변수: var(variable) / 상수: constant / let

var:

  • ES5(이전)에 사용
  • 현재는 let을 더 많이 사용, 호환성 유지로 사용은 가능
  • 함수 스코프를 가짐
  • 함수 내에서 선언되었다면 함수 블록 내에서만 유효
  • 호이스팅 발생(변수가 선언되기 전에 사용되어도 오류 발생 x)

let:

  • ES6(2015)에 도입
  • 블록 스코프를 가짐
  • 블록 내에서 선언된 변수는 블록 내에서만 유효
  • 변수 선언 한 번만 가능
  • value를 바꿀 수 있음
  • 호이스팅 발생(위에 참조)
  • TDZ(Temporal Dead Zone)라는 개념에 따라 선언 전에 접근 시, ReferenceError 발생

const:

  • ES6(2015)에 도입
  • 블록 스코프를 가짐
  • 상수(constant) 선언 때 사용
  • 한 번 할당 시, value 변경 x
  • 선언과 동시에 value를 할당해야 함
  • 호이스팅 발생(위 참조)
  • TDZ(Temporal Dead Zone)라는 개념에 따라 선언 전에 접근 시, ReferenceError 발생

차이점

  • var: 함수 스코프, 재선언 가능, 재할당 가능, 호이스팅 발생
  • let: 블록 스코프, 재선언 불가능, 재할당 가능, 호이스팅 발생, TDZ에 의한 접근 오류
  • const: 블록 스코프, 재선언 불가능, 재할당 불가능, 초기화 필요, 호이스팅 발생, TDZ에 의한 접근 오류
